Type Casting in Java

  • Type casting is used to convert an object or variable of one type into another.
  • Syntax. dataType variableName = (dataType) variableToConvert;
  • Notes. There are two casting directions: narrowing (larger to smaller type) and widening (smaller to larger type)

Program :

class TypeCast{
        public static void main(String args[]){
                 byte b,c;
                 int i=258;
                double d=323.142;
                System.out.println("\033[H\033[2J");
                System.out.println("Conversion of integer to byte : ");
                b=(byte)i;
                System.out.println("Integer And Byte value is "+i+" "+b);
                System.out.println("conversion double to int ");
                i=(int)d;
                System.out.println("double and int value is "+d+" "+i);
                System.out.println("Conversion of Double to byte is");
                c=(byte)d;
                System.out.println("Double to byte is "+ d+" "+c);
        }
}
